جدول شامل لجميع وسوم CSS

الوسم الوصف
colorتحديد لون النص.
font-sizeتحديد حجم الخط.
font-weightتحديد سماكة الخط (عادي، غامق).
text-alignمحاذاة النص (يمين، يسار، وسط).
text-decorationإضافة تأثيرات على النص مثل التسطير.
backgroundخاصية مختصرة لتنسيق الخلفية.
background-colorلون الخلفية.
background-imageوضع صورة كخلفية.
background-positionتحديد موضع الصورة.
background-repeatتحديد تكرار الخلفية.
background-sizeتحديد حجم الخلفية.
background-attachmentتحديد ما إذا كانت الخلفية ثابتة أو تتحرك.
borderتحديد الحدود حول العنصر.
border-radiusتحديد تدوير زوايا العنصر.
border-top-left-radiusتدوير الزاوية العلوية اليسرى.
border-top-right-radiusتدوير الزاوية العلوية اليمنى.
border-bottom-right-radiusتدوير الزاوية السفلية اليمنى.
border-bottom-left-radiusتدوير الزاوية السفلية اليسرى.
marginتحديد الهوامش الخارجية.
paddingتحديد الهوامش الداخلية.
widthتحديد عرض العنصر.
heightتحديد ارتفاع العنصر.
displayتحديد طريقة عرض العنصر (block, inline).
floatمحاذاة العنصر (يمين، يسار).
positionتحديد موضع العنصر (static, absolute, relative, fixed).
opacityتحديد شفافية العنصر.
box-shadowإضافة ظل حول العنصر.
outlineإضافة حدود خارجية للعنصر دون التأثير على حجمه.
visibilityإظهار أو إخفاء العنصر (visible, hidden).
@keyframesإنشاء رسوم متحركة مخصصة.
animationتحديد إعدادات الرسوم المتحركة.
animation-nameربط العنصر بحركة معينة.
animation-durationتحديد مدة تشغيل الرسوم المتحركة.
animation-timing-functionتحديد نوع التوقيت (ease, linear).
animation-iteration-countعدد مرات تكرار الرسوم المتحركة.
cursorتحديد شكل المؤشر عند تمرير الماوس.
pointer-eventsتحديد ما إذا كان العنصر يمكن التفاعل معه.
appearanceتحديد مظهر العنصر حسب نظام التشغيل.
