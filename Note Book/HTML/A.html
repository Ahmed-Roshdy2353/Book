لأنه يُستخدم في انشاء الروابط التي تتيح للمستخدمين التنقل بين الصفحات المختلفة أو داخل نفس الصفحة أو حتى للوصول إلى مستندات وملفات HTML أحد أهم عناصر <a> يُعتبر عنصر

<a> أهم السمات الخاصة بعنصر

( href ) -:1

: تستخدم هذه السمة لتحديد وجهة الرابط و يمكن أن تكون الوجهة كالتالي

  • . ( https://Google.com ) رابط خارجي مثل
  • . ( ../../HTML/Note Book.html ) ( Note Book.html ) رابط داخلي مثل
  • . ( section like (id) (#name )) مرجع داخلي في نفس الصفحة
  • . ( mailto:ahmedxxxxxxxxxxxxx@gmail.com ) رابط بريد إلكتروني مثل
  • . ( tel:+02010xxxxxxxx ) رابط هاتفي مثل
<a href="https://Google.com"> زيارة موقع خارجي </a>

( target ) -:2

تُحدد هذه السمة كيفية فتح الرابط

  • . يفتح الرابط في نفس علامة التبويب ( _self )
  • . يفتح الرابط في علامة تبويب جديدة ( _blank )
  • . يفتح الرابط في الإطار الأب ( _parent )
  • . يفتح الرابط في الإطار الأعلى ( _top )
<a href="https://Google.com" target="_blank">فتح في صفحة جديده </a>

( rel ) -:3

تُستخدم لتحديد العلاقة بين الصفحة الحالية والصفحة الهدف القيم الشائعة

  • . تمنع الصفحة المفتوحة من الوصول إلى الصفحة الأصلية (لتحسين الأمان) ( noopener )
  • . تمنع إرسال أي معلومات من الصفحة الأصلية إلى الصفحة الهدف ( noreferrer )
  • . تُخبر محركات البحث بعدم تتبع هذا الرابط ( nofollow )
<a href="https://Google.com" target="_blank" rel="noopener noreferrer">رابط آمن </a>

( title ) -:4

تُظهر نصًا توضيحيًا عند تمرير مؤشر الفأرة فوق الرابط

<a href="https://Google.com" title="هذا رابط إلى موقع Example">رابط مع عنوان </a>

( download ) -:5

تسمح بتنزيل الملف بدلاً من فتحه مباشرة. يمكن أيضًا تحديد اسم الملف

download
<a href="file.pdf" download="myFile.pdf">تحميل ملف PDF</a>

( hreflang ) -:6

تُحدد لغة المحتوى المستهدف

<a href="https://Google.com" hreflang="en">رابط لمحتوى باللغة الإنجليزية</a>
